# spring frmaework MVC게시판 & 로그인

### 1. 개발환경:  
   + OS : Windows 10  
   + Spring framework version : 4.3.22RELEASE  
   + Java version : 1.8  
   + WEB Server : Apache  
   + WAS Server : Tomcat 8.0  
   + ORM : mybatis 3.5.5  
   + DB : mariadb 10.5.5  

### 2. 기능:  
   #### 1.MVC게시판/crud    
  + 리스트(페이징처리)   
  + 글쓰기    
  + 글읽기    
  + 글수정   
  + 글삭제  
  + 댓글작성    
  + 댓글수정   
  + 댓글삭제    
  + 검색    
  
  #### 2.회원    
  + 회원가입    
  + 회원수정    
  + 회원탈퇴    
  
   #### 3.로그인/로그아웃    
  + 사용자 로그인/로그아웃
